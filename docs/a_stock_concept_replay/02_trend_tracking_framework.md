# 02 趋势追踪框架

> 本文用于判断新概念是否可能从普通题材发展为阶段主线。具体记录模板见 [03 追踪记录模板](03_tracking_templates.md)。

## 概念上涨的三种核心逻辑

| 上涨逻辑 | 核心观察 | 典型概念 | 主要风险 |
|---|---|---|---|
| 估值扩张 | 市场相信未来空间变大，但业绩尚未兑现 | 互联网+、元宇宙、AI应用早期、低空经济早期 | 业绩迟迟不兑现，估值坍塌 |
| 业绩增长 | 收入、利润、订单真实释放 | 新能源车、锂电、光伏、CXO、AI算力部分公司 | 增速放缓，供给过剩，估值消化 |
| 供需反转 | 价格、库存、产能周期改善 | 供给侧改革、猪周期、化工周期、光伏/锂电反转 | 反转不持续，供给再次释放 |

实际大行情往往是多因素叠加：政策或技术提供叙事，产业数据提供验证，资金环境放大涨幅。

## 概念生命周期模型

| 阶段 | 典型信号 | 投资者常见心理 | 主要风险 |
|---|---|---|---|
| 萌芽期 | 政策、技术、海外映射或产业新闻出现 | 少数人关注，市场半信半疑 | 把普通新闻误判为大主题 |
| 发酵期 | 龙头启动，新闻频率提升，板块开始异动 | 想参与但不确定 | 逻辑未验证，容易一日游 |
| 主升期 | 板块扩散，成交放大，机构覆盖增加 | 共识形成，趋势强化 | 估值快速抬升 |
| 高潮期 | 全民讨论，细分题材乱飞，利好钝化 | 害怕错过，追补涨 | 高位接盘 |
| 退潮期 | 龙头滞涨，补跌，业绩不及预期 | 从相信到怀疑 | 把趋势反转误判为回调 |
| 分化期 | 少数公司继续兑现，多数回落 | 重新寻找真龙头 | 错把反弹当二波 |

## 如何判断概念进入主升期

满足越多，越值得从观察池升入确认池：

1. 龙头率先走强，并且不是单日脉冲；
2. 板块内多只股票跟随，形成扩散；
3. 成交额、换手率持续放大；
4. 政策、产业、海外映射或业绩数据持续出现新催化；
5. 券商、基金、媒体开始系统性讨论；
6. 回调时龙头抗跌，利好出现后板块仍能继续上涨。

## 如何判断概念退潮

常见退潮信号：

1. 龙头高位放量滞涨或跌破中期趋势；
2. 利好消息不再推动上涨，甚至高开低走；
3. 板块从龙头上涨变成低位补涨、杂毛乱飞；
4. 估值极端，但业绩没有同步兑现；
5. 产业数据转弱，例如价格下跌、订单放缓、库存上升；
6. 政策边际变化或监管约束增强；
7. 机构重仓股开始连续下跌，基金净值压力加大；
8. 舆论热度极高，但新增资金接力不足。

## 三层概念池

| 层级 | 含义 | 操作方式 |
|---|---|---|
| 观察池 | 有政策、新闻、技术苗头，但市场还没充分反应 | 每周记录事件和代表公司 |
| 确认池 | 龙头放量上涨、板块扩散、产业数据开始验证 | 加密跟踪，研究产业链和龙头 |
| 风险池 | 已经大涨、估值高、舆论热、利好钝化 | 重点观察退潮信号，避免追高 |

## 追踪周期

| 频率 | 做什么 | 目的 |
|---|---|---|
| 每周 | 记录新政策、新事件、板块涨跌幅、成交额、龙头表现、新闻频率 | 捕捉趋势是否发酵 |
| 每月 | 判断概念池升级/降级 | 避免信息堆积但没有判断 |
| 每季度 | 对照财报、订单、毛利率、收入增速、机构持仓 | 验证概念是否从题材变成业绩 |
| 重大事件后 | 单独更新政策、产业大会、龙头业绩爆雷、海外映射大涨大跌 | 防止关键拐点被固定周期错过 |

## 日常只记录五类信息

日常每周只记录：新事件、龙头表现、板块扩散、产业验证、风险信号。

原则：

- 每个概念单周更新不超过 3—5 行；
- 只有进入确认池的概念才写研究卡；
- 连续 2—3 次没有新增验证的概念，降级或删除；
- 模板服务于判断，不是为了收集信息而收集信息。

## 不同类型概念的最佳观察方式

| 类型 | 最该看什么 | 最容易误判什么 |
|---|---|---|
| 政策驱动 | 政策级别、资金支持、试点落地、地方订单 | 把口号当订单 |
| 产业驱动 | 渗透率、订单、产能、价格、利润率 | 只看需求不看供给扩张 |
| 技术驱动 | 产品成熟度、成本下降、量产节奏、商业化场景 | 把技术演示当商业化 |
| 海外映射 | 国内公司收入占比、客户关系、供应链位置 | 把海外大涨等同于国内受益 |
| 事件驱动 | 事件持续性、需求是否一次性、政策是否临时 | 把短期需求当长期趋势 |
| 周期反转 | 库存、价格、产能出清、供需缺口 | 把短反弹当大周期反转 |

## 如何从新闻判断是否值得入观察池

看到一条新概念新闻时，问 6 个问题：

1. 是否指向足够大的产业空间？
2. 是否有上市公司能明确受益，而不是概念很远？
3. 是否可能持续出现后续催化，而不是一次性事件？
4. 是否已有龙头或板块开始异动？
5. 是否能找到后续验证指标，例如订单、价格、招标、装机、用户数、收入占比？
6. 是否存在一票否决风险，例如强监管、商业模式不成立、纯蹭热点？

如果前 5 条满足 3 条以上，且没有明显一票否决风险，可以进入观察池。

## 成功大主题与短命题材打分表

每项 0—2 分，总分 20 分。

| 维度 | 观察问题 | 评分参考 |
|---|---|---|
| 政策强度 | 是否有国家级、部委级或地方密集政策支持 | 0 无；1 局部；2 强政策 |
| 产业空间 | 市场规模是否足够大 | 0 小；1 中等；2 大空间 |
| 渗透率阶段 | 是否处于 0到1 或 1到10 扩张期 | 0 成熟；1 局部增长；2 高成长 |
| 技术/产品变化 | 是否有真实技术突破或成本下降 | 0 无；1 有进展；2 明显突破 |
| 订单/业绩验证 | 是否能进入上市公司收入利润 | 0 无；1 初步；2 明确兑现 |
| 龙头清晰度 | 是否有明确受益公司 | 0 不清晰；1 有候选；2 龙头明确 |
| 板块扩散 | 是否从少数个股扩散到产业链 | 0 无；1 局部；2 明显扩散 |
| 资金共识 | 成交额、ETF、机构覆盖是否提升 | 0 无；1 初步；2 强共识 |
| 估值位置 | 是否还没进入明显泡沫阶段 | 0 极高；1 偏高；2 合理 |
| 风险约束 | 政策、商业模式、供需、竞争风险是否可控 | 0 高风险；1 中等；2 可控 |

| 总分 | 判断 |
|---|---|
| 0—7 | 多半是短炒或噪音 |
| 8—13 | 可进入观察池 |
| 14—17 | 可能成为阶段主线 |
| 18—20 | 具备大主题潜力，但仍需动态验证 |

## 观察概念与实际投资标的研究必须区分

| 层面 | 重点 |
|---|---|
| 概念层面 | 空间、政策、产业趋势、市场共识、生命周期阶段 |
| 个股层面 | 收入占比、利润弹性、估值、竞争壁垒、治理质量、财务风险 |
| 风险点 | 很多股票只是名字相关，业务占比很低，属于蹭概念 |

## 从概念到个股研究的 5 步

1. **确认概念是真趋势还是短炒**：用打分表和生命周期判断。
2. **拆产业链**：上游材料/设备、中游制造、下游应用、服务平台。
3. **找真正受益环节**：看收入占比、利润弹性、供需格局、客户关系。
4. **筛代表公司**：龙头、弹性标的、卖铲人、平台型公司分别看。
5. **做风险排除**：估值过高、业务占比低、财务质量差、政策风险、竞争恶化。

## 如何识别龙头

| 龙头类型 | 判断标准 |
|---|---|
| 市场龙头 | 率先启动、涨幅领先、回调抗跌、成交额大、能带动板块 |
| 基本面龙头 | 收入占比高、利润弹性大、客户/技术/产能优势明显、财务质量好 |
| 最优标的 | 市场龙头和基本面龙头重合 |
| 风险标的 | 只有市场热度但基本面弱，或只有基本面但市场不认可 |

## 概念扩散路径

1. 龙头公司先涨；
2. 核心供应链跟涨；
3. 上游材料、设备、零部件扩散；
4. 下游应用和场景扩散；
5. 名字相关、边缘业务、低位补涨股被挖掘；
6. 最后进入泛概念化。

越往后，风险越高。

## 不同市场环境下的概念表现差异

| 市场环境 | 容易上涨的概念 | 风险 |
|---|---|---|
| 流动性宽松、风险偏好高 | 高弹性、远期想象空间、科技成长 | 容易泡沫化 |
| 经济弱、政策托底 | 政策主题、央国企、稳增长、红利资产 | 持续性取决于政策力度 |
| 产业景气上行 | 能兑现业绩的成长板块 | 估值提前透支 |
| 通缩/需求弱 | 高股息、低估值、防御板块 | 题材股持续性差 |
| 海外科技大周期 | 海外映射链、供应链公司 | 国内受益程度容易被高估 |

## 不同阶段最容易犯的操作错误

| 阶段 | 最容易犯的错误 |
|---|---|
| 萌芽期 | 因为没有上涨就忽略，或者只看新闻不做验证 |
| 发酵期 | 把一次异动当成主线，过早下重判断 |
| 主升期 | 只看涨幅不看估值、拥挤度和业绩兑现 |
| 高潮期 | 被全民讨论带动，追入边缘补涨股 |
| 退潮期 | 把趋势反转误判为正常回调 |
| 分化期 | 不区分真龙头和蹭概念公司，反复参与弱反弹 |

## 概念热度来源可信度分级

| 可信度 | 来源 | 用法 |
|---|---|---|
| 高 | 政策原文、上市公司公告、财报、订单、招投标、监管文件 | 用于确认事实和验证逻辑 |
| 中 | 权威财经媒体、产业会议、券商研报、机构调研纪要 | 用于发现趋势和补充线索 |
| 低 | 社交媒体热搜、短视频、论坛传言、自媒体标题党 | 只作为情绪温度计，不作为买入依据 |

## 趋势追踪删除机制

| 情况 | 处理方式 |
|---|---|
| 连续 2—3 次周复盘没有新增催化或验证 | 降级 |
| 连续 2 次月复盘核心逻辑没有强化 | 归档 |
| 龙头跌破趋势且无产业数据支撑 | 移入风险池 |
| 发现上市公司真实受益很弱 | 删除或改为警示案例 |
| 政策方向逆转、监管明确压制 | 直接降级或归档 |
| 事件驱动型需求结束 | 归档为短命题材复盘 |

## 如何处理已经大涨的概念

1. 判断它处于生命周期哪一段；
2. 看龙头是否仍在创新高，还是只剩补涨股活跃；
3. 看产业数据是否继续强化，还是只有情绪扩散；
4. 看估值是否已经极端，未来业绩是否能解释当前市值；
5. 如果是主升中段，重点研究龙头和核心环节；
6. 如果是高潮后段，转入风险池，避免把热度当机会。

## 复盘过去十年的趋势识别规则

1. 大级别行情通常不是单一新闻驱动，而是政策、产业、资金、业绩至少两项共振；
2. 只有宏大叙事、没有上市公司业绩承接，容易短命；
3. 代表龙头的持续性比概念股数量更重要；
4. 大主题往往从少数龙头走出，再扩散到产业链，最后扩散到边缘题材；
5. 当市场开始炒“名字相似、逻辑很远”的标的时，往往进入后期；
6. 真正强的趋势会在回调后继续创新高，弱题材通常只有一波流；
7. 政策驱动要看订单，技术驱动要看量产，产业驱动要看利润；
8. 事件驱动不能按长期成长股估值；
9. 产业趋势正确，不代表股价阶段正确；
10. 趋势追踪系统最重要的不是记录更多，而是及时升级、降级和删除。
