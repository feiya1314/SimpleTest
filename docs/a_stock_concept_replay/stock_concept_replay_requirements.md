# A股热门概念复盘与趋势追踪需求确认

## 目标

整理最近十年中国 A 股热门股票概念的兴起、持续、退潮过程和原因，并进一步形成一套可长期维护的趋势追踪系统，用于观察未来可能形成大涨的概念和板块。

说明：内容用于市场历史复盘和研究框架，不构成投资建议；代表股票仅用于说明当时市场认知和行情映射，不代表推荐。

## 已确认范围

- 时间范围：2015—2025/2026。
- 市场范围：以 A 股为主。
- 主题粒度：使用大众能理解的一级/二级主题，避免过细的短炒分支。
- 排列方式：按大致兴起时间顺序排列。
- 概念数量：控制在 20—30 个核心概念。
- 代表股票：每个概念列 3—6 个代表公司，按当时市场认知选取。
- 持续时间：主字段写主升行情周期，原因栏补充完整热度周期。
- 结束时间：写主升行情结束/退潮期；仍在进行的概念写“尚未完全结束，处于分化/验证期”。
- 原因深度：兴起原因、持续原因、结束原因要更详细，每个概念保留关键原因。

## 表格要求

拆成两张核心表：

1. 总览表：概念、驱动类型、主升期、退潮期、代表股票。
2. 详细复盘表：兴起原因、持续原因、结束原因。

驱动类型可包括：

- 政策驱动
- 产业驱动
- 技术驱动
- 流动性驱动
- 事件驱动
- 海外映射驱动

## 必须覆盖的分析维度

- 为什么当时会涨。
- 涨多久，主升期是什么时候。
- 为什么后来不涨或退潮。
- 是否是估值扩张、业绩增长、供需反转，或多因素叠加。
- 是否属于成功大主题或短命题材。
- 对未来识别类似概念有什么启发。

## 趋势追踪系统要求

追踪周期：以中短期主题趋势 + 中期产业验证为主，观察周期约 3 个月—2 年。

方法：自上而下发现主题 + 自下而上验证龙头。

概念池分层：

| 层级 | 含义 | 操作方式 |
|---|---|---|
| 观察池 | 有政策、新闻、技术苗头，但市场还没充分反应 | 每周记录事件和代表公司 |
| 确认池 | 龙头放量上涨、板块扩散、产业数据开始验证 | 加密跟踪，关注趋势持续性 |
| 风险池 | 已经大涨、估值高、舆论热、利好钝化 | 重点观察退潮信号，避免追高 |

## 追踪指标

- 早期信号：政策、产业订单、技术突破、海外映射、资本开支、龙头放量。
- 中期确认：业绩兑现、机构调研、产业链扩散、指数/ETF 成交放大。
- 高潮风险：全民讨论、估值极端、细分题材乱飞、利好钝化、高位放量滞涨。

信息来源优先级：

- 政策文件/重要会议
- 产业新闻/招投标/订单公告
- 上市公司公告/财报/互动平台/调研纪要
- 板块指数/ETF/成交额/涨停数量
- 券商研报标题和覆盖频率
- 海外映射
- 社交媒体热度，仅作为情绪温度计

更新频率：

| 频率 | 做什么 | 目的 |
|---|---|---|
| 每周 | 记录新政策、新事件、板块涨跌幅、成交额、龙头表现、新闻频率 | 捕捉趋势是否开始发酵 |
| 每月 | 判断概念池升级/降级 | 避免信息堆积但没有决策 |
| 每季度 | 对照财报、订单、毛利率、收入增速、机构持仓 | 验证概念是否从题材变成业绩 |
| 重大事件后 | 单独更新政策、产业大会、龙头业绩爆雷、海外映射大涨大跌 | 防止关键拐点被固定周期错过 |

### 行情属性与退潮性质

- 区分“行情主线”和“阶段性题材”。
- 结束原因区分正常退潮、逻辑证伪、事件结束。
- 正常退潮表示产业仍存在，但估值、增速、供需或风险偏好变化导致行情退潮。
- 逻辑证伪表示商业模式、业绩承接或政策环境与原叙事明显不符。
- 事件结束表示疫情、防疫药、口罩检测等事件驱动需求回落。

### 不同类型概念的观察方式

加入不同类型概念的最佳观察方式：

- 政策驱动：看政策级别、资金支持、试点落地、地方订单，避免把口号当订单。
- 产业驱动：看渗透率、订单、产能、价格、利润率，避免只看需求不看供给扩张。
- 技术驱动：看产品成熟度、成本下降、量产节奏、商业化场景，避免把技术演示当商业化。
- 海外映射：看国内公司收入占比、客户关系、供应链位置，避免把海外大涨等同于国内受益。
- 事件驱动：看事件持续性、需求是否一次性、政策是否临时，避免把短期需求当长期趋势。
- 周期反转：看库存、价格、产能出清、供需缺口，避免把短反弹当大周期反转。
