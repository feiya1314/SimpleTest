# [CLAUDE.md](http://CLAUDE.md)

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

IT技术面试问答笔记库，面向中高级开发岗位（阿里、腾讯、字节等），以面试问答形式组织技术知识点。当前覆盖 Java 和 MySQL 两个技术大类。

## 文档结构

- 顶层目录按技术大类编号命名：`01-Java/`、`04-Mysql/`
- 每个大类下为主题 md 文件，编号格式：`01-主题名.md`
- `主题规划.md`：大类与主题的索引表，新增主题时需同步更新
- `CHANGELOG.md`：变更记录，每次文档操作后需更新

## 文档格式规范

- 全文只有 # 一级标题,不包含"原理分析"等子标题
- 问题按 # 1. # 2. 顺序编号
- 问题之间模拟面试追问场景，保持逻辑递进，环环相扣
- 原理分析中关键概念和结论使用 **加黑** 突出显示
- 问题之间不使用 `---` 分隔
- 扩展问题作为追问的后续问题，与主问题关联
- 问题来源必须基于真实面试场景

## 操作约束

- 只处理 `.md` 格式文件
- 不删除已有文件（除非用户明确要求）
- 不生成测试代码、不执行编译或运行
- 新增大类时创建对应的编号目录
- 问题对应的回答要求要面向技术面试时的回答,无冗余书面语，全部可直接面试口述
