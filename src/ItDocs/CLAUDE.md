# [CLAUDE.md](http://CLAUDE.md)


## 项目概述

IT技术面试问答笔记库，面向中高级开发岗位（阿里、腾讯、字节等），以面试问答形式组织技术知识点。

## 文档结构

- 顶层目录按技术大类编号命名：`01-Java/`、`04-Mysql/`
- 每个大类下为主题 md 文件，编号格式：`01-主题名.md`
- `主题规划.md`：大类与主题的索引表，新增主题时需同步更新
- `CHANGELOG.md`：变更记录，每次文档操作后需更新

## 文档格式规范

- 全文只有 # 一级标题,不包含子标题
- 问题按 # 1. # 2. 顺序编号
- 原理分析中关键概念和结论使用 **加黑** 突出显示
- 问题之间不使用 `---` 分隔
- 问题来源必须基于真实面试场景
- 回答中涉及多个并列要点时，必须使用列表格式（`- `）逐条列出，不要将多项内容用分号拼接在同一行

## 回答要求
- 必须分析验证回答是否准确，不能给出错误的回复
- 如果问题下给出了参考信息，必须分析参考信息，保留参考信息的关键内容，如有图片需要保留。调整格式，去处冗余的内容。
- 如果问题下的参考信息不足以回答问题，请根据你的知识补充完问题的答案

## 操作约束

- 只处理 `.md` 格式文件
- 不删除已有文件（除非用户明确要求）
- 不生成测试代码、不执行编译或运行
- 新增大类时创建对应的编号目录
- 问题对应的回答要求要面向技术面试时的回答,无冗余书面语，全部可直接面试口述
